# Code of Conduct

This project follows a professional, respectful participation standard.

Expected behavior:

- Keep discussion focused on the technical issue or contribution.
- Give actionable feedback and accept review in good faith.
- Respect different use cases, editors, operating systems, and coding environments.
- Avoid publishing private information, credentials, logs, local paths, or repository structure from private projects.

Unacceptable behavior includes harassment, personal attacks, discriminatory language, sexualized language or imagery, and publishing private information without permission.

Project maintainers may edit, close, or remove issues, pull requests, comments, or wiki edits that violate this standard.
